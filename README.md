## 📖 About The Project

**India Story Project (ISP)** is a cinematic, story-driven mobile experience that brings India's history, culture and heritage to life. It pairs a **Flutter** mobile app with a **TypeScript / Node.js** backend and a dedicated **design system**, delivering an immersive, bilingual (EN / HI) journey through the stories that shaped a nation.

This repository — **`IndiaStoryProjectApp`** — is the mobile companion app of the wider India Story ecosystem, engineered for smooth performance, elegant motion, and a native cross-platform feel across Android, iOS, and the web.

> "Every monument has a memory, every memory deserves a story." 🏛️

## 🚀 Getting Started

### Prerequisites
```bash
✔ Flutter SDK (>=3.x)
✔ Node.js (>=18.x)
✔ Git
```

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/pradhummandil/IndiaStoryProjectApp.git
cd IndiaStoryProjectApp

# 2. Set up the backend
cd backend
npm install
npm run dev

# 3. Set up the mobile app
cd ../mobile
flutter pub get
flutter run
```
